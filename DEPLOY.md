# 🚀 Quick Deployment Guide

## ✅ সহজ Deployment (Vercel Dashboard)

1. **Vercel এ যান:** https://vercel.com/new

2. **GitHub Repository Import করুন:**
   - Select: `Mehedi259/Oman-sheba-website`

3. **Important Settings:**
   ```
   Root Directory: apps/web  ⚠️⚠️⚠️ (Must set this!)
   Build Command: pnpm build
   Install Command: pnpm install
   ```

4. **Deploy বাটনে click করুন** 🚀

---

## 📱 Result

Deployment URL: `https://oman-sheba-website.vercel.app`

---

**⚠️ Monorepo Setup: Root Directory MUST be `apps/web`**

**✨ That's it! 🎉**

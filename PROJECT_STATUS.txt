🚀 চালু করার নির্দেশনা:
======================

1. Dependencies install:
   pnpm install

2. Docker services চালু করুন:
   docker-compose up -d

3. Database setup:
   pnpm db:generate
   pnpm db:migrate
   pnpm db:seed

4. Development server:
   pnpm dev

5. Browser-এ খুলুন:
   http://localhost:3000
